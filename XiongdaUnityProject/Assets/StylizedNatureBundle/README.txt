购入素材「Stylized Nature Bundle」在本工程中的位置
================================================

1) 正式 Unity 资源包（请用菜单导入，不要指望仅复制此文件夹即生效）
   路径（工程根目录，不在 Assets 下）：
   PackagesToImport/StylizedNatureBundle.unitypackage

   Unity 菜单：Assets → Import Package → Custom Package… → 选上述文件 → Import。

2) 卖家商品预览图（已复制进工程，可作参考）
   Assets/StylizedNatureBundle/VendorPreviewImages/（1.jpg … 8.jpg）
   另有一张 PNG 预览：VendorPreview_effect.png（由购入素材中的效果图复制并重命名）。

3) 原始购入目录（含视频教程）
   XiongdaUnityProject/background/ 下的子文件夹（名称因系统编码可能显示异常，以资源管理器为准）。

若命令行导入失败，日志常见原因：Unity 编辑器已打开本工程，请先全部退出再导入。
